# Wanderlust - Travel Booking & Blog Platform

## Setup Instructions

### Backend
cd backend
npm install
npm start

   Server runs on `http://localhost:5000`.

### Frontend
cd frontend
npm install
npm run dev

   App runs on `http://localhost:5173`.

## Features
- **Packages**: Browse and book travel packages.
- **Blogs**: Read user travel stories.
- **AI Chatbot**: Get travel assistance.
- **Auth**: User registration and login.
